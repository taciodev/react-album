### 💻 Sobre o Projeto
Construção de álbum de fotos em React, colocando em prática os conceitos de react router e axios e consumindo uma API de fotos.
### 🛠 Tecnologias 
As seguintes ferramentas estão sendo usadas na construção do projeto:

* React
* CSS Modules
* TypeScript
* React Router
* Axios
